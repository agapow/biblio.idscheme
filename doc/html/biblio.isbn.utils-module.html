clean_isbn(isbn_str)

source code 

Strip whitepsace and punctuation from an ISBN representation.

For example:

>>> clean_isbn ("3-8055-7505-X")
'380557505X'
>>> clean_isbn (" 0-940016-73-7  ")
'0940016737'
>>> clean_isbn ("978-0-940016-73-6")
'9780940016736'
Parameters:
  • isbn_str (string) - The ISBN as a string, e.g. " 978-0-940016-73-6 "
Returns:
A normalised ISBN, e.g. "9780940016736"

isbn10_to_sbn(isbn_str, cleanse=True)

source code 

Convert an ISBN-10 to an old-style SBN.

For example:

>>> isbn10_to_sbn ("0-940016-73-7")
'940016737'
>>> isbn10_to_sbn ("0940016737", cleanse=False)
'940016737'
>>> isbn10_to_sbn ("1-56414-682-0")

>>> isbn10_to_sbn ("0-940016-73-7", cleanse=False)
Traceback (most recent call last):
        ...
AssertionError: input '0-940016-73-7' is not 10 digits

This is done simply by cropping a leading 0. Invalid ISBNs are simply returned as None.

Parameters:
  • isbn_str (string) - An ISBN-10 as a string, e.g. " 0-940016-73-6 ". It should be 10 digits after normalisation.
  • cleanse (boolean) - If true, formatting will be stripped from the ISBN before conversion.
Returns:
A normalised SBN, e.g. "940016736", or None if no conversion is possible.

sbn_to_isbn10(sbn_str, cleanse=True)

source code 

Convert an old-style SBN to an ISBN-10.

The conversion from an SBN to ISBN-10 is trivial (add a '0' prefix), but is provided for orthogonality.

For example:

>>> sbn_to_isbn10 ("940016-73-7")
'0940016737'
>>> sbn_to_isbn10 ("940016737", cleanse=False)
'0940016737'
Parameters:
  • sbn_str (string) - An SBN as a string, e.g. " 940016-73-6 ". It should be 9 digits after normalisation.
  • cleanse (boolean) - If true, formatting will be stripped from the ISBN before conversion.
Returns:
A normalised ISBN-10, e.g. "0940016736".

Exceptions:

AssertionError
If the input isn't a 9-digit SBN.
